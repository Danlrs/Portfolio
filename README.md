## 🇺🇸 English

A modern developer portfolio created to showcase my projects, technical skills, and experiences. Focused on a strong visual identity and responsiveness.

### 🚀 Technologies Used

This project was built with the following technologies:

- **[React](https://reactjs.org/)** - JavaScript library for building user interfaces
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **Vanilla CSS** - Custom styling built from scratch, without frameworks
- **Lucide React & React Icons** - Icon libraries for the interface

### 🛠️ How to Run Locally

Follow these steps to run the project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Danlrs/Portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
   *Access it in your browser (usually at `http://localhost:5173/`)*

### 📁 Structure

- `src/components/`: Main application components (e.g., `Navbar`, `About`, `Projects`, `Footer`).
- `src/assets/`: Static files and images.
- `src/index.css` & `src/App.css`: Global styles and design system.
